Filtering Options (Client-side / Additional Args)

Note: Netstat's built-in filtering capabilities vary by OS. For advanced filtering by state, port, or program name, it's often more effective to pipe the output to tools like `grep` (Linux/Termux) or `findstr` (Windows). These options will be reflected as comments in the generated command or require manual addition to 'Additional Arguments'.
